A set is a collection of unique items, 

A set goes through a list and picks the unique items without duplicates. A set ensures you have an oversight of everything in your set at a glance. 

A set is useful in checking the membership. i.e if an item is in the collection.  A set is listed by a curly bracket {} and separated by a comma , 

A dictionary on the other hand is a collection of items majorly, key-value items. It is more of a store, to access a value inside the store, you need a specific key that only the value responds to. Think in terms of atm machine has money in it. To access your own particular money, you input your atm card and enter your pin, this then brings out the value you want- money. 

A dictionary is equally listed by a curly bracket {} however, it separated by a : # hands-on
